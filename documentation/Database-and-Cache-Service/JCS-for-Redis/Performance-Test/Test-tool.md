# 测试工具

此次测试Redis性能使用的工具为Redis-benchmark 6.0版

## Redis-benchmark 6.0版简介
Redis_benchmark 6.0版是Redis Labs近期推出的命令行工具，可用于在键值存储数据库中生成数据负载并进行压力测试。
