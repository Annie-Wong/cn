# 短信使用短网址

### 业务场景

短信中对字数控制较严格，一条短信内容如果超出，将变成两条或者多条短信发送，成本将大大提升，如何在单条短信中传递更多的内容给客户端，是一个很重要的问题。

### 解决方案

使用短网址，能在有限的字符数量中传递更多的信息，为文案提供更多的发挥空间，客户端展现也将更加整洁和优美。

### 具体实现

- 通过平台统一发布相似或相同类短信给大量用户，包含相同短网址，发送频率较低，如1~2次/天

  可通过页面生成短网址，复制到短信发送平台的内容中进行发送，简单、快速、便利。

- 针对不同用户发送定制类短信，包含不同短网址，发送频率较高，几百~几万次/天

  通过API进行接口调用，使用SDK接入，简便快捷，与快速与短信系统对接使用。
