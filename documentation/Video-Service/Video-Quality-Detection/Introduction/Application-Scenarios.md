# 应用场景

**视频网站/媒体/短视频/电商/教育等**

视频网站/媒体/短视频/电商/教育等行业拥有海量的音视频内容需要面向C端用户，音视频审核、用户观看体验等环节及其重要，视频质量检测可实现对其海量音视频内容进行7x24小时不间断检测，大大降低人工审核成本，提升用户观看体验。且京东云为客户提供控制台和开放的API等接入方式，接入方便，减少了开发对接成本。
