# 产品概述

物联网智能边缘计算，是部署和运行在边缘节点设备上的智能系统，将云端计算能力扩展到本地，实现边缘自治。用户可以在现场通过边缘计算节点获得例如设备接入管理，数据就近处理，以及AI推断等能力。物联网智能边缘计算产品必须依赖物联网引擎产品共同使用。

物联网智能边缘计算 具备以下特性：

- 支持用户在边缘节点连接和管理子设备，并将子设备数据上报至物联网引擎。
- 边缘应用一键部署，配置，删除等管理功能。
- 本地数据实时采集，实时处理，分析，AI学习、推断、识别。
- 支持离线工作，加快边缘决策速度。
- 边缘到云端的安全传输及数据本地加密。
