# 产品规格

使用物联网智能边缘计算产品服务，需要您在边缘节点上安装边缘核心系统，因此对边缘节点硬件环境有如下要求：

|    名称    |        规格        |
| :--------: | :----------------: |
|   处理器   |       x86-64       |
|    主频    |      ≥1.6GHz       |
|    内存    |        ≥2GB        |
|  硬盘空间  |       ≥64GB        |
|  操作系统  | Ubuntu 18.04及以上 |
| Docker版本 |      ≥v18.06       |

**提示：如您需要使用边缘应用：对象检测服务，请您根据您实际用量，扩展本地硬盘空间。**
