# 限制说明

目前您可以免费使用物联网智能边缘计算产品服务，但需要注意以下限制：

| 类别         | 描述                                   | 限制 |
| ------------ | -------------------------------------- | ---- |
| 边缘节点数   | 单个账号下可以创建的最大边缘节点数量   | 50万 |
| 子设备数     | 单个边缘节点下可以关联的最大子设备数量 | 2000 |
| 边缘应用数   | 单个边缘节点允许部署的最大边缘应用数   | 30   |
| MQTT消息大小 | 单条MQTT消息最大长度                   | 64KB |
