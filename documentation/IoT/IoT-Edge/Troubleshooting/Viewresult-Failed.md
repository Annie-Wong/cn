# 无法查看对象检测应用的计算结果

## 问题描述

我在我自己的对象存储中无法看到对象检测应用的计算结果？

## 问题处理
 
1. 由于计算结果需要上传至您自己的对象存储中，因此需要您确认对象存储服务正常。

3. 请确认您访问的对象存储路径和您在边缘应用页面指定的路径是否一致。

1. 请确认您的对象检测应用已经正常运行，摄像头正常开启。

4. 完成上述几步后，请重新启动计算模块。

5. 详细操作步骤请您参考[边缘视觉检测实践](../Best-Practices/Object-Detect.md)。

## 后续处理
