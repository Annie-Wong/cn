# 打包部署
通过sca cli的简单命令，您可以将本地项目打包部署至云端。
## 部署命令

` sca deploy`     根据指定的函数模板配置文件，将配置文件中的指定代码包、函数配置等信息，打包部署到云端。

## 参数说明

| 简写 | 参数          | 必填 | 描述                                                   | 示例        |
| ---- | ------------- | ---- | ------------------------------------------------------ | ----------- |
| -t   | template-file | N    | 函数项目的部署用配置文件，支持 yaml 格式文件。         | deploy.yaml |


备注：当函数已存在时，强制更新并覆盖原有函数
