# 产品概述
京东云云容灾产品通过与公有云和私有云架构的对接，提供完整的IT恢复环境；从而保证灾难恢复服务的整体高可靠性。通过灵活高效的数据复制技术，提供关键数据的实时保护； 通过自动化管理平台，实现灾备全环境监控以及一键式切换操作，提供业务级和数据级的灾难恢复解决方案。

## 产品特性
 - 能够接入主流的公有云、私有云平台，支持混合云架构
 - 通过图形化的管理页面，简单直观地对灾备预案进行有效的配置和管理
 - 提供应用级云容灾，灾备全环境监控以及一键式切换操
 - 云容灾服务全生命周期管理，日常灾备、演练、切换、紧急运营和回切等整个灾备闭环的全部阶段
 - 提供灵活的服务，从数据级保护的云备份服务，到完整端到端应用级灾备云服务以及灾备自动化管理服务，提供一揽子灾备解决方案
