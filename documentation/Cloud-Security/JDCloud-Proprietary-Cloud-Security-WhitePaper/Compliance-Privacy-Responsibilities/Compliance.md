# 安全合规/隐私保护/权责归属

##	1.安全合规

京东云在努力完善业界标准认证体系的同时，致力于建立高效的安全内控体系，务求更好的向用户展示京东云的合规实践。

京东云权威安全资质认证：

- ISO27001 信息安全管理体系国际认证

- 公安部 “信息系统安全三级等级保护认证”

- 中国信息通信研究院 “可信云服务认证”

- 工业和信息化部 “ITSS云计算服务能力标准符合性证书”

-	支付卡行业数据安全标准认证(PCI DSS)

- CSA STAR云安全认证

- C STAR云计算安全评估认证
