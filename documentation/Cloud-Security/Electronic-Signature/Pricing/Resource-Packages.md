# 计费说明

## 购买须知

- 资源包生效期内，账单结算时，系统将按照免费额度 > 资源包 > 按量计费的顺序进行结算，超出免费额度和资源包部分将采用按量计费方式结算。
- 资源包购买后将会立即生效，将在下次推送账单时进行抵扣相关用量费用。
- 若您同时叠加购买了多个资源包，在抵扣相关用量费用时将会叠加抵扣。
- 用户账户发生欠费（账户余额小于0）时，无论是否处于资源包生效期，电子签章服务都将在欠费后停止服务。
