## 接入VPN

由于数据加密服务实例的限制，暂不提供公网访问，故需要您将您的电脑设备通过VPN接入到京东云VPC环境。

[限制说明](../Introduction/Restrictions.md)

## 设置白名单

由于数据加密服务实例的限制，京东云数据加密服务实例禁止一切访问，需要您将您的管理设备添加到白名单中。
