# 入门指南概述
## 开通流程
客户开通使用京东智联云边缘托管服务流程如下：

1. 售前需与您沟通确认具体需求；

2. 售前和产品根据您的需求确认所需的资源；

3. 销售与您签订完成合同并确认需求信息；

4. 京东智联云将您的资源信息录入后台，客户就可以在京东智联云官网控制台开始使用京东智联云边缘托管服务（与边缘物理计算服务共用一个控制台）。

5. 您可以通过控制台-边缘物理计算服务-工单管理模块，提交所需工单服务，方便快捷。
